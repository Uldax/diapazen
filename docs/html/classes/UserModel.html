      Connexion de l'utilisateur

      connectionToApp(string $email, string $password, string $login_ip) : \tableau

      On test tout d’abords si l’email et le password sont bien renseignés, puis avec une requête Select on récupère les infos en faisant un Where sur le password. Puis si on a un résultat et que le password passé en paramètre correspond à celui de la base de donnée, on met à jour la dernière connection grâce à updateConnectionDate et on renvoie un tableau avec toutes les données.

      Parameters

      $email

      string

      email email renseigné par l'utilisateur

      $password

      string

      password mot de passe renseigné par l'utilisateur

      $login_ip

      string

      login_ip Adresse IP de l'utilisateur

      Returns

      \tableauavec toutes les infos si tout s’est bien passé et false sinon
